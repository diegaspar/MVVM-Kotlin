# MVVM-Kotlin-Koin:rocket:

This is a project made just for fun to try the MVVM architecture.

## Libraries used
- Kotlin :heart:
- Koin (DI)
- MVVM (Architecture) and DataBinding
- Room (Persistence)
- Retrofit (Network)
- RxAndroid + RxJava2 (Reactive)
